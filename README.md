# SmartNotes DevOps Project

Student-level DevOps project based on INT332 syllabus.

## Tech Stack
- Spring Boot
- PostgreSQL
- Docker
- Docker Compose
- Maven
- GitHub Actions
- Jenkins
- AWS EC2

## Run Project

```bash
docker compose up --build
```

Frontend:
http://localhost:3000

Backend:
http://localhost:8080/notes
